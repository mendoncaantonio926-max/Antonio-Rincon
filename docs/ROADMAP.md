# Roadmap Imediato

## Produto

- consolidar billing/trial com transicoes de estado mais proximas do fluxo comercial real
- ampliar modulo de adversarios com comparacao mais profunda entre monitorados e leitura historica
- evoluir a camada de IA para sumarizacao ainda mais contextual por fluxo e acao recomendada

## Engenharia

- ampliar o lint/padrao automatizado para backend e outras rotas do monorepo alem do escopo atual em Biome
- ampliar cobertura automatizada para fluxos frontend alem do build, lint e do browser audit atual
- preparar estrategia de migracao futura de SQLite local para banco dedicado

## Operacao

- endurecer o fluxo de release com menos passos manuais e mais automacao guiada
- revisar protecoes de branch e regras de merge no GitHub
- endurecer seguranca de segredos e configuracao por ambiente
